# ocrd_paddleocr

OCR-D wrapper for the PaddleOCR framework.

**Work in progress.**

## Installation

For GPU use:

```python
python -m pip install paddlepaddle-gpu==3.2.0 -i https://www.paddlepaddle.org.cn/packages/stable/cu129/
python -m pip install .
```

For CPU use:
```python
python -m pip install paddlepaddle==3.2.0
python -m pip install .
```
